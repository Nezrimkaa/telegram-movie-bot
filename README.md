# 🎬 Telegram Bot "Что посмотреть?"

Telegram бот с подборкой лучших фильмов и встроенными видеоплеерами на русском языке!

## ✨ Возможности

- 🎥 **20 популярных фильмов** с русскими плеерами
- 📺 **Видео с VK** — встроенные плееры без рекламы
- 🔍 **Поиск фильмов** по названию
- 🎲 **Случайный фильм** — если не знаете что выбрать
- 📊 **Топ по рейтингу** — лучшие из лучших
- 🎭 **Жанры** — фильтрация по категориям
- 📱 **Адаптивный дизайн** для мобильных

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка токена

Токен уже настроен. Для своего бота замените в `.env` или `bot.py`:

```python
BOT_TOKEN = "ваш_токен_от_BotFather"
```

### 3. Запуск

```bash
python bot.py
```

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и меню |
| `/movies` | Все фильмы со списком |
| `/search` | Поиск по названию |
| `/random` | Случайный фильм |
| `/top` | Топ-5 по рейтингу |
| `/genre` | Фильмы по жанрам |
