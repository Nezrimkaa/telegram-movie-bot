# 🎬 Telegram Bot "Что посмотреть?"

Telegram бот с подборкой лучших фильмов и **бесплатными плеерами без подписки**!

## ✨ Возможности

- 🎥 **20 популярных фильмов** с бесплатными плеерами
- ✅ **Без подписки** — смотри бесплатно!
- 📺 **RuTube + YouTube** — бесплатные источники
- 🔍 **Поиск фильмов** по названию
- 🎲 **Случайный фильм**
- 📊 **Топ по рейтингу**
- 📱 **Удобный интерфейс**

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка

Токен уже настроен. Для своего бота замените в `.env`:

```python
BOT_TOKEN = "ваш_токен_от_BotFather"
```

### 3. Запуск

```bash
python bot.py
```

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и меню |
| `/movies` | Все фильмы |
| `/search` | Поиск по названию |
| `/random` | Случайный фильм |
| `/top` | Топ-5 по рейтингу |
