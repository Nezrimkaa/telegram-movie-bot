# 🎬 Telegram Bot "Что посмотреть?"

Telegram бот с подборкой из **100 фильмов** с бесплатными плеерами без подписки!

## ✨ Возможности

- 🎥 **100 популярных фильмов** с бесплатными плеерами
- ✅ **Без подписки** — смотри бесплатно!
- 📺 **RuTube + YouTube** — бесплатные источники
- 🔍 **Поиск фильмов** по названию
- 🎲 **Случайный фильм**
- 📊 **Топ-10 по рейтингу**
- 📱 **Удобная клавиатура с кнопками**

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка токена

В файле `.env` замените токен:

```python
BOT_TOKEN = "ваш_токен_от_BotFather"
```

### 3. Запуск

```bash
python bot.py
```

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и меню |
| `/movies` | Все фильмы (100) |
| `/search` | Поиск по названию |
| `/random` | Случайный фильм |
| `/top` | Топ-10 по рейтингу |
